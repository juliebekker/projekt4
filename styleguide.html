Navigationsbare

På appen/hjemmesiden har vi to navigationsbare. En i toppen og en i bunden. Den i toppen har en hjem og tilbage knad. Den i bunden har kuponer, sociale, profil og samlemærker knapper. I stedet for ord, vises der ikoner i vores navigationsbare. Navigationsbaren i bunden er sticky, så den sidder altid i bunden af skærmen også selvom man scroller.

Top navigationsbar

Html

<nav class="navtop" id="navtop"> <ul> <li><a href="#"><img src="images/backarrowicon.svg" alt="Tilbage"></a></li> <li><a href="#"><img src="images/homeicon.svg" alt="Hjem"></a></li> </ul> </nav>

Css

.navtop { top: 0; width: 100%; background-color: #7499A3; } nav ul { position: relative; display: flex; justify-content: space-between; align-items: center; list-style: none; } nav ul li a img { width: 100px; height: 100px; margin: 30px; }

Bottom navigationsbar

Html

<footer> <nav class="nav" id="nav"> <ul> <li><a href="#"><img src="images/couponicon.svg" alt="Kupon"></a></li> <li><a href="#"><img src="images/friendsicon.svg" alt="Venner"></a></li> <li><a href="#"><img src="images/profilicon.svg" alt="Profil"></a></li> <li><a href="#"><img src="images/badgeicon.svg" alt="Badge"></a></li> </ul> </nav> </footer>

Css

.nav { position: fixed; bottom: 0; width: 100%; background-color: #7499A3; } nav ul { position: relative; display: flex; justify-content: space-between; align-items: center; list-style: none; } nav ul li a img { width: 100px; height: 100px; margin: 30px; }

Javascript

var nav = document.getElementById('nav'); window.onscroll = function () { if(window.pageYOffset > 100) { nav.style.position = "fixed"; nav.style.bottom = 0; }else { box.style.position = "absoulte"; box.style.bottom = 100; } }

Typografi

Typografien er taget ud fra ucl's design guide. Alt tekst har farvekoden #003A44 og er arial. Header 1 og 2 er bold, hvor normal brødtekst er normal i vægt.

Kode i css

h1 { font-weight: bold; color: #003A44; font-family: arial; } h2 { font-weight: bold; color: #003A44; font-family: arial; } p { color: #003A44; font-family: arial; }
