Farve koder

Farverne der bruges, er taget ud fra ucl's design guide.

Brug rgb farvekoden i 'background-color' eller 'color' i css fil

  • BAGGRUND #F1F8F8 (Columbia blue 20%)
  • IKONER #025560 (Midnight green 80%)
  • NAV & MENU #7499A3 (Midnight green 40%)
  • CTA #F58E76 (Terra cotta 100%)
  • LÅST #DDDDDD

Navigationsbare

På appen/hjemmesiden har vi to navigationsbare. En i toppen og en i bunden. Den i toppen har en hjem og tilbage knad. Den i bunden har kuponer, sociale, profil og samlemærker knapper. I stedet for ord, vises der ikoner i vores navigationsbare. Navigationsbaren i bunden er sticky, så den sidder altid i bunden af skærmen også selvom man scroller.

Top navigationsbar

Html

            <nav class="navtop" id="navtop">
              <ul>
                <li><a href="#"><img src="images/backarrowicon.svg" alt="Tilbage"></a></li>
                <li><a href="#"><img src="images/homeicon.svg" alt="Hjem"></a></li>
              </ul>
            </nav>

Css

            .navtop {
              top: 0;
              width: 100%;
              background-color: #7499A3;
            }
            nav ul {
              position: relative;
              display: flex;
              justify-content: space-between;
              align-items: center;
              list-style: none;

            }

            nav ul li a img {
              width: 100px;
              height: 100px;
              margin: 30px;
            }

Typografi

Typografien er taget ud fra ucl's design guide. Alt tekst har farvekoden #003A44 og er arial. Header 1 og 2 er bold, hvor normal brødtekst er normal i vægt.

Kode i css

            h1 {
              font-weight: bold;
              color: #003A44;
              font-family: arial;
            }

            h2 {
              font-weight: bold;
              color: #003A44;
              font-family: arial;
            }

            p {
              color: #003A44;
              font-family: arial;
            }

Javascript

            function openPage(pageName,elmnt,color) {
              let i, tabcontent, tablinks;
              tabcontent = document.getElementsByClassName("tabcontent");
              for (i = 0; i < tabcontent.length; i++) {
                tabcontent[i].style.display = "none";
              }
              tablinks = document.getElementsByClassName("tablink");
              for (i = 0; i < tablinks.length; i++) {
                tablinks[i].style.backgroundColor = "";
              }
              document.getElementById(pageName).style.display = "block";
              elmnt.style.backgroundColor = color;
            }
